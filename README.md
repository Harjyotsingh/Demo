# Neon Aura AR ✨

An advanced browser-based Hand Tracking AR experience built using MediaPipe Hands, HTML5 Canvas, and Web Audio API.

## Features

- Real-time hand tracking
- Neon glowing hand skeletons
- Gesture detection
- Pinch interaction effects
- Particle physics system
- Shockwave/ripple animations
- Dynamic themes
- Matrix-style animated background
- Audio-reactive effects
- Dual-hand interaction system
- Fully browser-based (No installation required)

## Technologies Used

- HTML5
- CSS3
- JavaScript
- MediaPipe Hands
- Canvas API
- Web Audio API
